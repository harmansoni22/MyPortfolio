# MyPortfolio

A clean, fast personal portfolio built with Next.js and Tailwind CSS.  
It’s the place I use to introduce myself, highlight the projects I’m proud of, and track how my work evolves as I move from higher secondary into college.

## About

I like building things that are small, sharp, and actually used.  
This portfolio focuses on:

- Projects that taught me something real (not just tutorial clones)
- Clear structure over visual noise
- A simple tech stack I can keep iterating on

The content will change as I grow, but the goal stays the same: show what I can do today, and make it obvious how I’m improving over time.

## Tech Stack

- **Framework:** Next.js  
- **Styling:** Tailwind CSS  
- **Language:** JavaScript  
- **Hosting:** Vercel
